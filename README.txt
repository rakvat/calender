latex-Ausgabe-Format (linux)
---------------------------
(Ausgabemodus 5)
erzeugt eine *.txt Datei im ausgabe-Verzeichnis
diese kann mit latex zu *.dvi Datei verarbeitet werden, die dann mit dvipdf zum pdf konvertiert wird




Benutzung des debug Modus:
mit Windows: Start-> Ausführen-> cmd 
             zum Verzeichnis calender\bin\debug wechseln
             calender.exe         startet das Programm im normalen Modus
             calender.exe debug   startet das Programm im debug Modus mit zusätzlichen Informationen
